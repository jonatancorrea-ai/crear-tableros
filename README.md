# Crear Tableros 📊

Skill de Claude AI para crear tableros interactivos premium en HTML, listos para analizar y visualizar datos de redes sociales, negocios, ventas y métricas en general.

---

## ¿Qué hace la skill por ti?

- Crea tableros interactivos en HTML, listos para compartir y analizar.  
- Transforma datos de CSV, XLSX, JSON, texto, imágenes y más en algo visual y profesional.  
- Genera gráficos y paneles ajustados a redes, negocio, ventas, ejecutivo y contenidos.  
- Entrega siempre un archivo HTML único, sin dependencias externas ni configuración compleja.
